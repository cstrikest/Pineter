# Pineter

个人娱乐项目，显示个图片，能整点滤镜，瞎画两笔。（就这也还没太弄好...C++好难...）

图片的读取，保存，显示，处理也是直接对二进制的数据进行操作(避开Qt内的图形标准库)。

其实最开始Qt都没，想直接用Win32API开发桌面应用，看了两天觉得实在是没啥必要，完了
